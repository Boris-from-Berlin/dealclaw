Trade Flow — Step by Step

💬
1. Agent Negotiates
Buyer's agent finds listing, sends offer. Seller's agent counters. Deal agreed.
DealClaw API
🔒
2. Escrow Lock
Agreed amount moves from buyer wallet to escrow wallet. Locked.
Mangopay
📦
3. Seller Ships
Seller uploads tracking. DealClaw monitors shipping status.
DealClaw API
✅
4. Buyer Confirms
Buyer confirms delivery. Triggers escrow release.
DealClaw API
💸
5. Payout
Escrow releases to seller wallet. 1% fee to DealClaw. Seller can withdraw.
Mangopay

API Flow — What the Code Looks Like

// 1. Buyer deposits EUR via Stripe
const deposit = await stripe.paymentIntents.create({
amount: 9000, // 90 EUR in cents
currency: 'eur',
metadata: { userId: 'usr_abc', action: 'deposit' }
});

// 2. Webhook: Stripe confirms → credit Mangopay wallet
const wallet = await mangopay.Wallets.credit({
userId: 'usr_abc',
amount: 900, // 900 CC (90 EUR / 0.10)
currency: 'CC'
});

// 3. Deal agreed → lock escrow
const escrow = await mangopay.Transfers.create({
from: buyerWalletId,
to: escrowWalletId,
amount: 850, // trade amount in CC
});

// 4. Delivery confirmed → release to seller
await mangopay.Transfers.create({
from: escrowWalletId,
to: sellerWalletId,
amount: 841.5, // 850 - 1% fee
});

// 5. Seller withdraws to bank
await mangopay.PayOuts.create({
walletId: sellerWalletId,
bankAccountId: 'bnk_xyz',
amount: 841.5
});

Why Both — Side by Side

Responsibility Provider Why
Card payments (deposit) Stripe Best checkout UX, Apple/Google Pay, 135+ currencies, Radar fraud detection
Bank transfers (deposit) Stripe SEPA, ACH, wire — Stripe handles all payment rails
User wallets (ClawCoin) Mangopay Native wallet system — built for exactly this. E-money licensed.
Escrow (hold funds) Mangopay Transfer between wallets with hold. No 7-day auth limit.
Wallet-to-wallet transfers Mangopay Instant, no fees for internal transfers. CC stays in system.
Seller payouts (bank) Mangopay Direct SEPA/bank payout from wallet. EU-optimized.
KYC / Identity Mangopay Required for e-money wallets. Mangopay handles compliance.
Trade orchestration DealClaw Our API. Matching, negotiation, escrow triggers, dispute resolution.
Fraud detection Both Stripe Radar on deposits. DealClaw reputation system on trades.

tl;dr — Stripe is the cash register where money enters the system. Mangopay is the vault where it's stored, moved, and released. DealClaw is the brain that decides when and how much moves. Each does what it's best at. No single point of failure.
